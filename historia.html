Instalación y uso de Android Studio

Descarga e instalación

Para comenzar, descarga Android Studio desde el sitio oficial de Android Developers. Ejecuta el instalador y sigue el asistente de instalación, seleccionando los componentes recomendados: SDK de Android, emulador y herramientas adicionales.

Durante la instalación puedes definir la ruta de instalación y configurar las opciones iniciales. Al abrir Android Studio por primera vez, se descargarán las dependencias necesarias para tu entorno.

Creación de un proyecto

Una vez instalado, puedes crear tu primer proyecto desde File → New → New Project. Selecciona el tipo de aplicación, el lenguaje de programación (Kotlin recomendado, también soporta Java) y la versión mínima de Android que deseas soportar.

Estructura básica del proyecto

Android Studio organiza los proyectos en módulos, siendo el más común el módulo app. Dentro de este módulo encontrarás:

  • Manifests: archivo AndroidManifest.xml con permisos, actividades y servicios.
  • Java/Kotlin: código fuente principal y pruebas.
  • Resources (res/): layouts, imágenes, íconos y valores como textos y colores.
  • Gradle Scripts: configuración de dependencias, versiones y compilación.

Ejecutar y probar la aplicación

Android Studio incluye un emulador que permite probar tu aplicación en diferentes dispositivos virtuales. También puedes conectar un dispositivo físico mediante USB para ejecutar la app directamente.

Las herramientas de depuración permiten seguir el flujo de tu código, establecer puntos de interrupción y corregir errores rápidamente. Además, con Android Profiler puedes monitorear el rendimiento en tiempo real (CPU, memoria y batería).
